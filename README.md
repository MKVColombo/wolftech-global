# WolfTech Spring Boot Clone

A responsive, original Spring Boot/Thymeleaf recreation inspired by the visual system and interaction style of WolfTech Global.

## Run

Requires Java 8+. The included Maven wrapper downloads the pinned Maven version automatically.

```powershell
.\mvnw.cmd spring-boot:run
```

Open `http://localhost:8080`. Run the test suite with `.\mvnw.cmd test`.

## Included

- Animated systems boot sequence and responsive cyber-industrial landing page
- Service modules, manifesto, radar/contact and systems footer sections
- Accessible mobile navigation and reduced-motion support
- Validated `POST /api/contact` endpoint with an interactive contact dialog
